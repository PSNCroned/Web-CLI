back

Moves backward in history

> back

bottom/bot

Scrolls to the bottom of the page

> bot

cgo

A contextual go command that navigates to a predefined label based on the current site

> go youtube
> cgo subs
Additional arguments can sometimes be passed, such as when navigating to a specific user page:
> go twitter
> cgo user google

close

Closes the current tab

> close

down/d

Scrolls down, can specify a factor by how much to scroll

> down
or
> d 2

dup

Duplicates the current tab

> dup

forward/f

Moves forward in history

> f

go

Navigates to a particular url or label

> go https://google.com
or
> go google
Additional arguments can sometimes be passed, such as when navigating to a specific page within a website:
> go wikiarticle google

help

Opens this help page in a new tab

> help

img

Searches the given query on Google Images

> img banana

r

Refreshes the current tab

> r

reset

Resets Web CLI, clearing all settings and saved macros/scripts

> reset

search/s

Searches the given query on Google

> s banana

set

Set command substitutions and custom lables for go and cgo.

> set cmd helpme help
Typing helpme would then have the same effect as help.
> set go mysite https://mysite.com
Typing go mysite would then navigate you to https://mysite.com
> set go goog google
Typing go goog would then be equivalent to go google.
> set cgo home /home
Typing cgo home while on the same site would take you to /home.
> set cgo s subs
Typing cgo s while on the same site would be equivalent to cgo subs.
> set cgo user /user/{0}
Typing cgo user john while on the same site would take you to /user/john.
> set cgo search /search?q={*}
Typing cgo search sample search words while on the same site would search for 'sample search words' on that site

tab

Allows for switching between tabs

> tab
Makes a new tab
> tab f
Goes to the first tab
> tab l
Goes to the last tab
> tab n
Goes to the next tab
> tab p
Goes to the previous tab
> tab 2
Goes to the second tab

top

Scrolls to the top of the page

> top

up/u

Scrolls up, can specify a factor by how much to scroll

> u
or
> up 3

wait

Waits the specified amount of miliseconds. Useful for command chains and macros.

> wait 500
Waits 500 miliseconds (half a second)

wiki

Go directly to a wikipedia article

> wiki banana

window/win

Creates a new window

> win

-t

If the command involves navigation, performs the navigation in a new tab.

> go google -t

-f

If the command makes a new tab, does not focus on the tab.

> go google -t -f

*n

Repeats the preceding command n number of times.

> go google -t *2
Opens two new tabs and navigates them to https://google.com
> close *3
Closes three tabs in succession

Command chains and macros

You can perform multiple commands in succession by separating them with a semicolon. If a command involves creating or changing focus to a tab, the next command will occur in the newly created or focused tab.

> go youtube -t; cgo subs
or
> tab 3; go google

Instead of needing to type these command chains out each time, you can save them as a macro using the extension icon in the top right. The name of the macro then becomes a useable command. For instance, if you saved go youtube -t; cgo subs under the name ytsubs you could use

> ytsubs

Scripts

You can even write your own JavaScript for your own custom commands. After clicking the extension icon in the top right, you can save JavaScript code under any command name and have that code run every time you enter that command.

Your script runs in the context of the extension's content script itself, meaning you have access to variables like args which is an array that contains the arguments entered with the command.

You also have access to the store(), retrieve(), and deleteKey() functions, which allow you save and retrieve data in the extension's background page like so: 

	//Data is saved in a JavaScript object

	store("test", 5); //Stores 5 under the key 'test'

	retrieve("test", function (data) { //Retrieves the value stored under key 'test'
		//callback
	});

	retrieve("test", function () { //Retrieves the entire storage object
		//callback
	});

	deleteKey("test"); //Delete the key 'test' from the storage object
